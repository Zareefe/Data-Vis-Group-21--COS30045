Offences over time

Annual mobile phone fines from all available years. Each line represents a jurisdiction. Use this view to assess long-term improvements or increases in enforcement.

Key observations

  • Lines trending downward indicate jurisdictions with improving offence rates.
  • Hover over the chart to see exact values for each year and jurisdiction.
  • Use the focus dropdown to highlight a specific jurisdiction.

Analysis notes

This trend analysis helps identify which jurisdictions demonstrate consistent improvement (reductions in fines) over time and which show increasing enforcement activity. In your report, you can quantify improvement using year-over-year percentage change and discuss possible reasons for these trends.

Data source: BITRE police enforcement fines dataset (mobile phone use offences).
